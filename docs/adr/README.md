## ADRのフォーマット

各ADRは以下の構成で記述しています：

1. **議論の背景**: 決定が必要になった理由、要件、制約
2. **選択肢と結論**: 検討した選択肢の一覧と最終結論
3. **各選択肢の比較表**: 複数の観点での比較
4. **結論を導いた重要な観点**: 決定の根拠となった主要な考慮点
5. **帰結**: トレードオフ、影響、将来の見直し条件
6. **各選択肢の説明**: 各選択肢の詳細と採用/不採用の理由

## ステータスの定義

| ステータス | 説明 |
|-----------|------|
| 提案 | 議論中、まだ決定されていない |
| 採用 | 決定され、実装に移行可能（または実装済み） |
| 廃止 | 別の決定により置き換えられた |
| 保留 | 追加情報待ち、または優先度低下 |

## 新しいADRの作成
